1. 簡介

成績發送平台（以下簡稱「本服務」）是由 陳子祥（以下稱「開發者」）所開發的開源工具。本服務致力於保護使用者的隱私權，本政策旨在說明我們如何收集、使用及保護您的資訊。

2. 我們收集的資訊

本服務僅在您授權的情況下，透過 Google API 存取以下特定資訊：

  • Google 帳戶資訊 (User Info)：僅存取您的電子郵件地址，用於確認操作者身分及顯示於介面右上角。
  • Google 試算表 (Google Sheets)：僅讀取您在介面中指定的試算表網址內容，用於取得成績與收件人資料。
  • Gmail 服務：使用您的 Gmail 帳號發送郵件給您指定的收件人。

3. 資訊的使用方式

我們收集的資訊僅用於以下目的：

  • 驗證您的身分以確保操作安全。
  • 讀取您指定的成績資料以生成通知信件。
  • 代表您寄送成績通知信給學生或家長。

4. 資料的儲存與分享

  • 不儲存資料：本服務為純前端與 Google Apps Script 運行之工具，所有資料處理皆在 Google 的伺服器記憶體中即時完成。開發者不會將您的試算表內容、學生個資或信件內容儲存於任何外部伺服器或資料庫中。
  • 不分享資料：我們不會將您的資訊出售、交易或轉讓給任何第三方。

5. 有限使用要求 (Limited Use Disclosure)

本服務使用從 Google APIs 取得的資訊將遵守 Google API Services User Data Policy，包括有限使用 (Limited Use) 要求。
